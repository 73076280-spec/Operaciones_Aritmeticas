# Operaciones Aritméticas

Este proyecto implementa las **cuatro operaciones aritméticas básicas**:  
**Suma, Resta, Multiplicación y División**.  

Se desarrolla en **Python**, incluyendo versiones de programación tradicional y **orientada a objetos**.

## Funcionalidades del Proyecto

- **Suma** de dos o más números.
- **Resta** entre dos números.
- **Multiplicación** entre dos números.
- **División** entre dos números (control de división por cero incluido).
- Versión **POO** (Programación Orientada a Objetos) que permite reutilizar el código fácilmente.
